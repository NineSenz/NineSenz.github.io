    # 一、service模块

    service模块：用于控制服务的启动,关闭,开机自启动等。

    https://docs.ansible.com/ansible/latest/modules/service_module.html#service-module

    参数 说明
    name 服务名称
    state reloaded, restarted, started, stopped 服务管理
    enabled yes|no 开启是否启动
